1. 메인페이지 작업개요
  1)  "도로명주소 안내시스템"
        메인페이지 레이아웃 작업
  2) 사이트 주소
      https://www.juso.go.kr/openIndexPage.do


2. 이미지 파일 사용하기
  1)  페이지에서 다운로드 받을 수 있는 자료는
       우클릭 후 직접 다운로드 받습니다.
  2) 다운로드 받을 수 없는 자료는
      개발자도구의 "Elements" 탭에서 
      해당하는 html요소를 찾은 후 
      "Styles" 탭을 확인하여 찾습니다.
  3) 위의 방식으로 찾을 수 없을 경우
     "Sources"탭을 클릭한 후 "Page"항목에서
     관련 자료를 검토하여 찾습니다.
      

3. 그외 관련 폴더, 파일 및 소스는 
   작업자가 판단하여 임의 제작합니다.


참고. 텍스트박스(Textbox) 태그
     => <input type="text" size="숫자만기재">
                                           ------------
                                        가로폭 크기설정
          또는 input선택자의 width를 사용하여
                 CSS에서 가로폭 크기 설정 가능
